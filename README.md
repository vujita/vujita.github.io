## Setup

Prequisites:

Install [nvm](https://github.com/nvm-sh/nvm)

---

[node](https://nodejs.org/en/) version `14.15.1`

Install [yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable).
If suggest using homebrew if you are on MacOS

```bash:
nvm install
yarn install
```
